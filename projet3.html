Projet 3

Présentation

Ce projet de site Internet porte sur une page web d'attente, en Working Progress.
Il propose un exemple de ce type de page, pour un site Internet d'actualité totalement fictif, Ducky Web News.
Ce site Internet fictif a la particularité d'avoir des contenus en français, en anglais et en allemand, mes trois langues.
Cette page est largement inspirée de celles de sites Internet de médias existants, comme LCI ou France Info par exemple, sans toutefois chercher à reproduire le même type d'interface. Le modèle choisi reprend leurs caractéristiques principales, tout en les simplifiant. Je n'ai pas voulu élaborer une page web trop complexe. Pour moi, elle doit avoir une présentation et une structure très claire, sans oublier les liens vers les réseaux sociaux.

Résumé

Ce site web comporte trois parties, les mêmes que pour un autre site web classique.
Le site web Ducky Web News, média imaginaire sur l'actualité des nouvelles technologies et de l'informatique, a un header avec deux menus. Le premier menu a des liens vers les jeux, les émissions en continu, les jeux, la newsletter et l'accessibilité. Le logo stylisé du média suit, avec le second menu qui donne accès aux différents types d'actualités classées par catégories et par univers tech.
La partie centrale se décomposé avec une photo centrale de montagne, sous laquelle un message d'attente pour les fidèles followers ou abonnés a été posté en anglais, en français et en allemand. Il s'agit de les faire patienter avant d'ouvrir l'accès vers un site web rénové.
Le footer est formé des mentions légales, avec en plus des accès vers des onglets vers un espace presse, un médiateur, la newsletter, l'expression des internautes.
