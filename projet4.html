Projet 4

Présentation

Le deuxième projet de site Internet fait en solo concerne les IDE, les éditeurs de texte. Les éditeurs de texte sont des outils indispensables pour le développeur web dans son travail quotidien de codage. Ils constituent des éléments importants de son poste de travail sur son ordinateur. C'est pourquoi il en utilise souvent plusieurs, les essaie, il se met au courant sur les nouveaux éditeurs de code. De nombreux éditeurs de code sont open source et peuvent être complétés par des extensions qui ajoutent de nouvelles fonctionnalités selon les besoins des développeurs.
C'est le cas des trois éditeurs de texte choisis, tous les trois gratuits : Visual Studio Code, Brackets et Atom. Ils ont tous les trois pour points communs d'être très adaptables, de rendre possible le codage sur un très grand nombre de langages de programmation. En outre, ils sont constamment actualisés et bénéficient d'une large communauté de développeurs, auteurs d'extensions et de nombreux tutoriels.
Ce projet est amené à s'élargir vers la présentation d'autres éditeurs de texte : en effet, ils sont très nombreux, et il en reste beaucoup à explorer.

Description

IDE : Page d'accueil

La page d'accueil de ce projet présente les trois éditeurs de texte choisis, en expliquant les raisons de leur choix.
Après une première partie consacrée à la présentation, la deuxième partie est divisée en trois colonnes, chaque colonne se rapportant à un IDE.
Chaque colonne comporte des paragraphes brefs sur l'éditeur de texte, une capture d'écran, un bouton de téléchargement vers le site Internet de l'éditeur. Les logos des éditeurs de code renvoient aux sites Internet des éditeurs. La barre de menu d'accueil permet sur un clic d'accéder aux pages dédiées.
Dans le footer, les icones mènent sur un clic vers les réseaux sociaux, sous la barre de menu sur les mentions légales.

Visual Studio Code

Visual Studio Code est l'éditeur de code open source conçu par Microsoft pour les développeurs web. Il possède de nombreuses extensions et fonctionnalités constamment mises à jour par une communauté de développeurs nombreuse. C'est celui que j'utilise le plus fréquemment à cause de sa facilité d'usage, d'Emmet intégré, de sa polyvalence. Ses diverses fonctionnalités permettent de personnaliser son outil en fonction de ses besoins. Il est aussi l'un des outils les plus appréciés actuellement par la communauté des développeurs.

Atom

Atom est l'éditeur de texte open source développé par Github. De même que Visual Studio Code et Brackets, il est polyvalent, multi-plateforme. Il a de très nombreuses fonctionnalités et extensions qui le rendent personnalisable, grâce à sa grande communauté de développeurs. Il est très flexible, très simple à manipuler, ce qui explique la forte croissance de ses utilisateurs et ce qui en fait un concurrent sérieux de Visual Studio Code. Leurs styles graphiques sont très proches, de même que leurs caractéristiques. C'est un outil fiable, agréable à utiliser et à connaître, d'autant plus que comme Visual Studio Code, il s'emploie autant pour le front end que pour le back end.

Brackets

Brackets est un éditeur open source à destination du développement web et du web design. Il cible plutôt les développeurs web en front end. Il a été créé et maintenu par Adobe. Il suppporte 38 langages de programmation. Comme les deux autres éditeurs précedemment cités, il a un riche vivier d'extensions que le développeur peut installer selon ses besoins et ses envies, pour faciliter son travail de codage. Lui aussi dispose d'une communauté de développeurs qui crée de nouvelles extensions, maintient les existantes à jour et publie des tutoriels. C'est un outil qui gagne à être connu, même s'il est plus efficace pour le développement web en front end.
